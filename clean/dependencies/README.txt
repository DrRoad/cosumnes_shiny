READ ME for updating the files in dependencies

elev.txt is important because it contains the calculation for the effective cable length for each well with a LevelSender installed. 
The manual measurements should be updated every 6 months, but this hasn't been done recently.

Email list contains the emails of people who want to receive the daily report

ll_battery_outages.txt and ls_battery_outages.txt contain the start and end date of missing data in the yyyy-mm-dd hh:mm format

ls_recovered_data.txt has the data taken from the leveloggers during the missing times with the date in the yyyy-mm-dd hh:mm format in the first column,
temperature in the second column in degrees C and level in meters in the far right column.
